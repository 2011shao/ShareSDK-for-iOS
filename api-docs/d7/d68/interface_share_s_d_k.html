followAccounts关注的微博账号信息，为字典结构，key为分享类型，value为ISSUserField对象(此属性仅用于Modal视图样式，并且仅在新浪微博和腾讯微博中有效)。
authViewStyle授权视图样式，参考SSAuthViewStyle枚举类型
