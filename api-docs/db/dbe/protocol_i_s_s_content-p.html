添加QQ空间内容单元，制定QQ空间分享时的内容使用此参数，如果参数设置为INHERIT_VALUE则默认使用父级类型相关参数

参数
title标题
url链接地址
site分享的来源网站名称，请填写网站申请接入时注册的网站名称。
fromUrl分享的来源网站对应的网站地址url
comment评论内容
summary分享分享内容概述
image分享图片
type内容类型，4表示网页；5表示视频
playUrl视频的swf播放地址，在type为5时必须填写
nswb是否同步到微博，等于1表示不同步，非1表示同步
